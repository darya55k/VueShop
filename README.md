# VueShop

## Описание проекта
Проект представляет собой интернет-магазин косметики, в котором пользователь может найти подходящий ему товар

## Функционал проекта
* Добавление товара в корзину

## Основные технологии:
* `HTML`, `CSS`
* `БЭМ`
* Работа с макетом в Figma
* Фреймворк `Vue.js`
* `Vuex`
* `Git`

 
## Дальнейшие планы
